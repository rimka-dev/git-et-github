Cours GIT

Qu'est que GIT

GIT est un outil de versionning et il permet d'avoir une traçabilite et un historique de tous les fichiers du projet

Il permet également de travailler en équipe: envoyer nos modifications à nos collégues et récuperer les modifications des collégues

Dans la traçabilité des fichiers GIT nous donne toujours 3 infos : qui, quand, pour quoi?

Installer et configurer git

git --version
git config --global user.email 'k.takka01@gmail.com'
git config --global user.name 'karim'

Commandes de base

initialiser un projet:

git init

le statut des fichiers

git status
ajouter des fichiers
git add 'nom du fichier'
Pour ajouter tous les fichiers:
git add .
Pour enregistrer définitivement une version du fichier on commit les fichiers
git commit
un commit nécessite toujours un message
Commit avec message :
git commit -m 'message'
commit avec add + message (commit avec add + message fonctionne sur les fichiers déja add précedamment') :
git commit -a -m 'message'

Correction des fichiers

Voir les différents commit:
git log
corrigier les modification sur un fichier pas encore add!
git checkout 'nom du fichier'
